# Solitaire - 接龍遊戲

## 1. 專案概述

- **專案名稱**：Solitaire (接龍)
- **類型**：網頁遊戲（經典紙牌遊戲）
- **玩法**：將牌按花色從 A 到 K 排列
- **目標**：清空所有牌

## 2. 遊戲規則

- 7 疊牌，最左 1 張向右遞增
- 只能移動最下面的牌
- 紅黑交錯排列
- 4 個收牌區，按花色 A-K 排列
- 牌堆可翻 3 張或 1 張

## 3. 技術規格

- **技術**：HTML5 + CSS3 + JavaScript
- **儲存**：LocalStorage（最高分數）
- **檔案結構**：
```
solitaire/
├── index.html
├── css/
│   └── style.css
└── js/
    └── game.js
```
