# AquaLife - 水族缸養魚遊戲

## 1. 專案概述

- **專案名稱**：AquaLife
- **類型**：網頁遊戲（單人休閒養成遊戲）
- **核心玩法**：玩家經營自己的水族箱，購買魚種、餵食、清潔、繁殖賺取金幣
- **目標用戶**：休閒玩家、喜歡養成遊戲的人

## 2. 視覺與美術風格

- **整體風格**：寫實風格
- **配色**：深海藍 (#0a1628) 為底色，配上柔和的光影效果
- **水波動效**：CSS 動畫模擬水面波紋
- **魚類圖案**：使用 Unicode emoji 或 SVG 寫實魚類
- **UI 设计**：玻璃質感面板，半透明背景

## 3. 遊戲系統

### 3.1 資源系統
| 資源 | 用途 |
|------|------|
| 💰 金幣 | 購買魚、飼料、裝飾品 |
| 💎 珍珠 | 高級貨幣（可購買稀有魚） |
| 📈 經驗值 | 升級水族箱 |

### 3.2 魚類系統
| 屬性 | 說明 |
|------|------|
| 名稱 | 魚的名字 |
| 價格 | 購買價格 |
| 成長時間 | 從幼魚到成鱼的時間 |
| 售價 | 卖出价格 |
| 壽命 | 魚的壽命（時間） |
| 飢餓度 | 需要餵食 |
| 幸福度 | 影響產金幣速度 |

### 3.3 魚種列表（完整版）
```
基礎魚類：
- 金魚 (Goldfish) - $50
- 孔雀魚 (Guppy) - $30

高級魚類：

### 3.4 水族箱系統
- 水質指數（0-100）
- 裝飾品（增加幸福度）
- 水草（增加水質）
- 濾水器（自動改善水質）

### 3.5 商店系統
- 魚類商店
- 飼料商店
- 裝飾品商店
- 魚缸升級

## 5. 技術規格

### 5.1 技術棧
- **前端**：HTML5 + CSS3 + JavaScript
- **儲存**：LocalStorage（遊戲存檔）
- **無需後端**：純客戶端遊戲

### 5.2 檔案結構
```
AquaLife/

├── css/
│   └── style.css   # 樣式
